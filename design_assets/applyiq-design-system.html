📐 Gradient Usage Rules
Use Brand gradient on primary CTA buttons and the sidebar logo area only.
Use Subtle gradient on page backgrounds and empty states.
Use Aurora / Purple→Pink on hero text (gradient text-clip trick) — sparingly.
Never stack gradients — if a button has a gradient bg, the text should be flat white.
Gradient Text Effect (CSS)
.gradient-text { background: var(--gradient-brand); -webkit-background-clip: text; -webkit-text-fill-color: transparent; background-clip: text; }

Motion & Easing

Hover over each card to see the easing in action.

ease-out (200ms)
standard UI
spring (350ms)
delightful moments
fade (120ms)
instant feedback
Motion Rules
fast (120ms): Hover states, focus rings, color changes — anything instant-feeling.
base (200ms): Card transitions, button presses, input focus — standard interactions.
slow (350ms): Modals opening, page transitions, kanban drag-drop landing.
spring: Only for delightful moments — confetti trigger, offer card, first AI analysis done.

Tailwind Config

Paste this into your tailwind.config.ts to get all tokens as utility classes.

import type { Config } from 'tailwindcss' const config: Config = { darkMode: ['attribute', '[data-theme="dark"]'], content: ['./src/**/*.{ts,tsx}'], theme: { extend: { fontFamily: { sans: ['DM Sans', 'system-ui', 'sans-serif'], mono: ['DM Mono', 'monospace'], }, colors: { purple: { 50: '#F5F3FF', 100: '#EDE9FE', 200: '#DDD6FE', 300: '#C4B5FD', 400: '#A78BFA', 500: '#8B5CF6', 600: '#7C3AED', 700: '#6D28D9', 800: '#5B21B6', 900: '#4C1D95', 950: '#2E1065', }, }, borderRadius: { sm: '6px', md: '10px', lg: '14px', xl: '20px', '2xl': '28px', }, boxShadow: { glow: '0 0 24px rgba(139,92,246,0.3), 0 4px 16px rgba(109,40,217,0.2)', card: '0 2px 8px rgba(109,40,217,0.06), 0 1px 2px rgba(0,0,0,0.04)', lifted: '0 8px 32px rgba(109,40,217,0.12), 0 4px 8px rgba(0,0,0,0.06)', }, animation: { shimmer: 'shimmer 1.5s infinite', toastIn: 'toastIn 0.3s cubic-bezier(0.34,1.56,0.64,1)', }, }, }, } export default config
